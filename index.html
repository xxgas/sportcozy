KEBIJAKAN RETURN

Kami memastikan setiap produk sampai dengan kondisi terbaik. Berikut kebijakan penukaran & pengembalian.

Syarat Pengembalian

  • • Produk belum dipakai dan masih dalam kondisi baru
  • • Label/tag masih terpasang
  • • Pengajuan maksimal 2x24 jam setelah barang diterima
  • • Menyertakan bukti foto/video unboxing

Tidak Berlaku Jika

  • • Salah memilih ukuran oleh pembeli
  • • Kerusakan akibat penggunaan pribadi
  • • Tidak ada bukti unboxing

Proses Return

  1. 1. Hubungi admin melalui WhatsApp
  2. 2. Kirim bukti foto/video produk
  3. 3. Tunggu konfirmasi dari tim kami
  4. 4. Kirim kembali produk sesuai instruksi
